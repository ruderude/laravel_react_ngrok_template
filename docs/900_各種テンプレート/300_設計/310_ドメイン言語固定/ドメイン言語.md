# ドメイン言語

**プロジェクト名**: （プロジェクト名を記載）

---

## 1. 本ドキュメントの目的

ドメインモデリングの初期段階で、ドメインに登場する言葉を洗い出すためのドキュメントです。
要件定義・ユーザーストーリーから抽出した候補を列挙し、後工程でユビキタス言語として整理します。

### 進め方

1. 要件定義・ユーザーストーリーから名詞・動詞・状態・ルールを抽出する
2. 同義語・類似語も削除せずすべて残す
3. ドメインエキスパートとレビューし、正式な用語を決定する
4. 決定した用語は「ユビキタス言語」ドキュメントに転記する
